# My Counting Web App

A simple counter app built with HTML, CSS, and JavaScript (jQuery).

## Features
- Count up or down with + and - buttons
- Set a custom goal
- Confetti animation triggers automatically when you hit your goal
- Clear button to reset the count

## Built With
- HTML
- CSS
- JavaScript (jQuery)
